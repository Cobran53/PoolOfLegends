  • If you inputed a custom patch, check your spelling. It should look like 13_9 or 13_10, with an underscore ! A patch can also be too old, so not on U.GG anymore. (PBE is not supported by this program.)
  • If you used "Latest", maybe the patch is so recent there isn't enough data. Try to input the previous patch in "custom". (This is especially true if your pick is rare-ish: Karthus Bot, Seraphine Mid...)
  • Check the puppet browser. It should have a mention like "Chrome/Edge is controlled by a program" or a robot icon in the search bar. If an error show up there, it's probably a U.GG problem. It could be fixed by simply reloading this page, or not. Contact me if the issue persists.
  • If you don't have the latest version of Pool of Legends, maybe the bug has been fixed? Go to the credits page to find the GitHub link.

If none apply, I have no idea what happened!
